# Experimental skin texture provenance

The source colour, normal, displacement, occlusion, and specular JPG files were
supplied by the project owner on 2026-07-16. The Warm, Medium, Deep, and Rich
colour maps are deterministic colour-only derivatives of the supplied Light
map; all pore placement remains aligned with the shared PBR maps. GNM Studio's
Neutral option derives a near-white grayscale colour map at runtime so users can
keep pore detail with no skin-tone tint; it does not add another binary asset.

The project owner confirmed on 2026-07-17 that the source texture set is
MIT-licensed. The original download URL and author/copyright line have not yet
been supplied. MIT redistribution requires preserving that copyright and
permission notice, so add the exact upstream attribution here before publishing
these files. Do not relabel the textures or their colour derivatives as
Apache-2.0.
